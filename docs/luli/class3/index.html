Homework

Complete the following excercises. When you are done, share the script with me! You have until February 21 (Monday) to hand me this assignment.

Lartip: If you don't know how to do something, try Googling it!

  1. (Number Comparison) Using the online Python IDE, create a script that:
    1. Asks the user for a number.
    2. Asks the user for another number number.
    3. Prints if the first number is bigger than the second (True or False).
      • For example, if the user enters first 15 and then 50, the program should print: Is 50 greater than 15? True.
    4. Prints if the first number is equal to the second (True or False).
      • For example, if the user enters first 15 and then 50, the program should print: Is 15 equal to 15? False.
  2. (Basic Adder) Using the online Python IDE, create a script that:
    1. Asks the user for a number.
    2. Asks the user for another number number.
    3. Adds both numbers.
    4. Prints the result.
  3. (Greeter) Using the online Python IDE, create a script that using only two variables:
    1. Asks the user their name.
    2. Asks the user their surname.
    3. Greets them.
      • For example, if the user enters Luli and then Spark, the program should print Hi there, Luli Spark!.
  4. (Substringer) Using the online Python IDE, create a script that using only one variable:
    1. Asks the user to enter a string (some text).
    2. Prints the first three letters of that string.
    3. After you've done that, can you solve the excersise without using ANY variables at all?
  5. (String search) Using the online Python IDE, create a script that using only one variable:
    1. Asks the user to enter a string (some text).
    2. Turns that string into lowercase.
    3. Checks if said string is the text the quick brown fox jumps over the lazy dog.
      • For example, if the user enters Potato, the program should print False, as the word potato is not contained in the string above.
      • For example, if the user enters Fox, the program should print True, as the word fox is not contained in the string above.
