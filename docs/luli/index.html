Luli's Python Challenge

Objectives

This will be the list of things you are expected to learn from this challenge and before you can start looking for your first Python Jr. developer job. I've written this in English, because being able to read, write and talk in English is a necessity in the software industry.

Basic Python

These are things you should learn to begin using the Python programming language. I believe this will be the hardest part, as you have to get the hang of the language and programming in general. You can do it! It's not that long :)

  1. Hello World, Basic Syntax, Comments and Variables
  2. Data Types, Numbers, Strings & Type Casting
  3. User Input, Booleans and Operators
  4. If/Else/Elif, While Loops and For Loops
  5. Lists, Tuples and Dictionaries
  6. Functions
  7. Scoping

Intermediate Python

This next section will teach you more advanced, but equally necessary, python topics, suchas as reading files or installing python libraries (or even, what libraries are!)

  1. Local Python Instalation
  2. Sets
  3. Classes & Objects
  4. Inheritance
  5. Basic libraries: time, datetime, math
  6. Pip
  7. Modules
  8. Try/Except
  9. File I/O

Future Topics

This last section will teach you a little about web scrapping and databases, that is knowledge you should have to have some leverage against other job candidates! Knowing these things is what helped me land my first jobs, so you can do it too!

  1. HTML & Beautiful Soup
  2. Basic SQL and MySQL I/O with Python
  3. Selenium & Web Automation
  4. Googling and Stack Overflow

When I started working, I didn't really know much more that what's on this list, so I believe it will be more than enough for you to find a starter job and kickstart your career <3 I'm more than sure that you can do it and start hunting for opportunities very soon!
